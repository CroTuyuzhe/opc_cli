# Tester Spec

## Test Case Format
```
TC-{id}: {title}
Precondition: ...
Steps:
  1. ...
  2. ...
Expected: ...
Actual: ...
Verdict: PASS / FAIL / BLOCKED
```

## Coverage Requirements
- All acceptance criteria from PRD must have test cases
- Edge cases: empty input, max length, special characters
- Cross-browser: Chrome, Safari, Firefox (if web)

## Bug Report Format
```
BUG-{id}: {title}
Severity: Critical / Major / Minor / Cosmetic
Steps to reproduce:
  1. ...
Expected: ...
Actual: ...
```
