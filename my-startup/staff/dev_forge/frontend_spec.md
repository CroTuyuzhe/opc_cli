# Frontend Development Spec

## Defaults
- HTML5 + CSS3 + Vanilla JS (unless user specifies framework)
- Mobile-first responsive design
- Semantic HTML structure

## When User Specifies Framework
- Follow framework best practices
- Use official recommended project structure
- Prefer composition over inheritance

## Code Quality
- No inline styles in production code (use CSS classes)
- All interactive elements must be keyboard accessible
- Images must have alt text
