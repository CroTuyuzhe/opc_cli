# Design Spec

## Visual Standards
- 8px grid system
- Color contrast ratio >= 4.5:1 (WCAG AA)
- Font sizes: body 14-16px, headings scale 1.25x

## Component Rules
- Buttons: min touch target 44x44px
- Form inputs: visible labels, clear error states
- Icons: consistent style (outline or filled, not mixed)

## Responsive Breakpoints
- Mobile: 320-767px
- Tablet: 768-1023px
- Desktop: 1024px+
